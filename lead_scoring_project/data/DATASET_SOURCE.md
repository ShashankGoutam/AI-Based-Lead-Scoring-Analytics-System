# Dataset Source

This project is based on the **Bank Marketing** dataset from the UCI Machine Learning Repository.

- UCI dataset page: https://archive.ics.uci.edu/dataset/222/bank+marketing
- Citation details and metadata are available on that page.

## File in this repository
- `bank.csv` is a small sample subset included for offline/demo use.
- For production-ready training/evaluation, download the full UCI dataset and replace this sample file before running `notebook/model_training.ipynb`.
